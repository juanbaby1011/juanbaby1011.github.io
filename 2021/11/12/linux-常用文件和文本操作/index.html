linux 常用文件和文本操作

发表于2021-11-12|更新于2021-11-12|linux
|阅读量:

# 文件操作

# touch 创建文件

创建指定名称后缀的文件

touch test
touch test.conf

# cp 复制

必须的目标文件名

cp test1.conf test2.conf

递归复制，包含子目录时用，不加 -r 的话，目录包含子目录文件的话，就会报错

cp -r origin dest

# mv 移动、重名命、备份

移动文件到 /tmp

mv test.conf /tmp

备份文件

mv test test.backup

排除指定目录

mv !(child1|child2) child1

如查报错

-bash: !: event not
执行下面命令解决
shopt -s extglob

# rm 删除文件

有提示删除

rm test1.conf

不提示，直接删

rm -f test.conf

删文件夹时加 - r, 不要提示则 -rf

rm -r dir

删除目录不提示 -f 不提示

rm -rf test/
